## GETTING STARTED

- Clone the repository

```bash
git https://github.com/huynhthikimtrung321/typescript-training.git
```

- Move to folder `practice`

```bash
cd TypeScript-Training/practice
```

- Edit the content of env.sample and rename it into .env

- Install packages

```bash
npm install
```

- Run the server

```bash
npm start
```
